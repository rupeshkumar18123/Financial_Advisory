## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/rupeshkumar18123/Financial_Advisory.git
   cd Financial_Advisory
````

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Launch the app:

   ```bash
   streamlit run app.py
   ```
